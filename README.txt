How to compile and run:
* Open cmd terminal and navigate to the folder containing the unzipped project files
* Run the Makefile with cmd 'make all'
* Run diskinfo with cmd './diskinfo <disk_name>.IMA'
* Run disklist with cmd './disklist <disk_name>.IMA'
* Run diskget with cmd './diskget <disk_name>.IMA <file_to_copy>'
* Run diskput with cmd './diskput <disk_name>.IMA <file_to_write>'

NOTE: all filenames and extentions written via diskput must be in all caps.
